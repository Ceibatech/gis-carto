# MULCV GeoArchives

Prototype web de plateforme nationale pour localiser, recenser, évaluer, prioriser et suivre les sites d'archives du ministère sur le territoire ivoirien.

## Modules couverts

- Carte SIG nationale avec filtres par région, statut et niveau de risque.
- Registre national des sites et indicateurs de volume documentaire.
- Questionnaire d'évaluation avec calcul des scores de risque et de priorité.
- Planification des vagues de mobilisation et chronologie opérationnelle.
- Espace de pièces justificatives et journal d'audit.

## Commandes

```bash
npm run dev
npm run build
```
